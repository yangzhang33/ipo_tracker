====================
一、产品目标
====================

实现一个面向个人使用的美国 IPO 数据跟踪系统。系统每天自动运行，完成以下任务：

1. 发现近期美国 IPO 候选公司
2. 同步 SEC EDGAR filings
3. 从 prospectus / filing 中提取 IPO 核心字段
4. 提取 lock-up 期限和预计到期日
5. 导出 3 份 CSV 报表

第一版只做：
- 美国普通 IPO
- 外国发行人通过 F-1 在美国上市的 IPO

第一版不做：
- SPAC
- direct listing
- ADR 二次上市
- ETF / 基金类产品
- OTC
- LLM
- 买卖打分
- 新闻情绪
- 实时行情
- 复杂财务估值

系统目标是“单纯的数据提取系统”。

====================

====================
七、字段口径要求
====================

第一版请固定以下口径，不要擅自复杂化：

1. free_float_at_ipo
= shares_offered_total

2. float_ratio
= free_float_at_ipo / shares_outstanding_post_ipo

3. lockup_start_date
- 如果有 424B4，则默认使用 424B4 filing_date
- 若没有 424B4，则退回使用包含 offer_price 的 prospectus filing_date

4. lockup_end_date
= lockup_start_date + lockup_days

5. 第一版不做复杂老股东提前流通识别
6. 第一版不做复杂 staged unlock 拆分，只要识别并标注即可

====================
八、状态机要求
====================

请实现 issuer 的简单状态机更新逻辑：

- candidate -> filed
  条件：出现 S-1 / F-1

- filed -> priced
  条件：出现 424B4 且 offer_price 非空

- priced -> trading
  第一版可以先不精确判断
  可选简化为 pricing_date 后 1 个交易日近似设为 trading
  或先不自动切到 trading，只保留 priced

- 任意 -> withdrawn
  条件：出现 RW
